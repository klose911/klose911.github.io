_事件_ 是 Redis 服务器的 *核心* ， 它处理两项重要的任务：
+ 文件事件：在多个客户端中实现 _多路复用_ ，接受它们发来的命令请求，并将命令的执行结果返回给客户端
+ 时间事件：实现服务器定时操作 _server cron job_
