*字典* _dictionary_ ， 又名 *映射* _map_ 或 *关联数组*  _associative array_ ， 是一种抽象数据结构， 由一组 *键值对* _key-value pairs_ 组成， 各个键值对的键各不相同， 程序可以 _添加_ 新的键值对到字典中， 或者基于键进行 _查找_ 、 _更新_ 或 _删除_ 等操作
