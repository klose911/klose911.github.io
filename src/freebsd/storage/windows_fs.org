FreeBSD 对
+ NTFS 的原生支持仅限只读，读写操作需借助用户态驱动 *ntfs-3g* 
+ FAT32 则由内核原生支持读写

* 目录结构

#+begin_example
  /
  ├── usr
  │   ├── local
  │   │   └── bin
  │   │       └── ntfs-3g              # NTFS-3G 可执行文件
  │   └── ports
  │       └── filesystems
  │           └── ntfs                  # ntfs-3g Ports 目录
  ├── etc
  │   ├── rc.conf                        # 系统启动配置文件
  │   └── fstab                          # 持久化挂载配置文件
  ├── dev
  │   ├── da0                            # 磁盘设备
  │   └── da0s1                          # NTFS 分区
  ├── media
  │   └── NTFS                           # NTFS 挂载点
  └── mnt
      ├── NTFS                           # NTFS 挂载点（备用）
      └── (通用挂载点)
#+end_example
