#+begin_example
  bhyve 是 FreeBSD 原生的虚拟机管理程序（hypervisor）

  通过 vmm 内核模块提供硬件辅助虚拟化支持，客户机操作系统可直接利用 CPU 虚拟化扩展指令集
#+end_example

*BVC* _bhyve Virtual-Machine Control Panel_ 是 bhyve 的 Web 图形化管理工具，可创建、启动、停止虚拟机

#+begin_example
  本节实验环境基于 FreeBSD 14.2-RELEASE。BVCP 2.1.4 版本尚不支持中文本地化

#+end_example
